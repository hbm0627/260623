# 서비스 개요

## 서비스명

개인 홈페이지 서비스

## 한 줄 소개

개인 소개, 회원 기능, 게시판, 댓글, 관리자 기능을 포함한 게이미피케이션형 개인 웹사이트.

## 서비스 목표

- 방문자가 운영자의 개성과 활동을 한눈에 파악할 수 있게 한다.
- 회원은 로그인 후 게시글, 댓글, 대댓글을 작성할 수 있다.
- 관리자는 어드민 계정으로만 접속해 유저가 생성한 게시물, 댓글, 대댓글을 관리할 수 있다.
- 귀여운 공룡 캐릭터와 석기시대 콘셉트로 일반적인 개인 홈페이지보다 기억에 남는 경험을 만든다.

## 목표 사용자

| 사용자 | 설명 |
| --- | --- |
| 비회원 방문자 | 메인 페이지, 게시글 목록, 게시글 상세를 둘러보는 사용자 |
| 일반 회원 | 로그인 후 게시글, 댓글, 대댓글을 작성하는 사용자 |
| 관리자 | 어드민 계정으로 로그인해 게시물과 댓글을 관리하는 운영자 |

## 페이지 구성

| 페이지 | 경로 예시 | 접근 권한 | 설명 |
| --- | --- | --- | --- |
| 메인 페이지 | `/` | 전체 | 개인 소개, 공룡 캐릭터, 최신 게시글, 주요 링크 표시 |
| 회원가입 페이지 | `/signup` | 비로그인 | 이름, 이메일, 비밀번호로 가입 |
| 로그인 페이지 | `/login` | 비로그인 | 일반 사용자 로그인 |
| 프로필 페이지 | `/profile` | 로그인 사용자 | 내 정보와 내가 작성한 게시글 확인 |
| 게시판 목록 페이지 | `/boards` | 전체 | 게시글 목록, 검색, 페이지네이션 |
| 게시글 상세 페이지 | `/boards/:id` | 전체 | 게시글 본문, 조회수, 댓글, 대댓글 |
| 게시글 작성 페이지 | `/boards/new` | 로그인 사용자 | 게시글 작성 |
| 게시글 수정 페이지 | `/boards/:id/edit` | 작성자 본인 | 본인이 작성한 게시글 수정 |
| 관리자 로그인 페이지 | `/admin/login` | 비로그인 | 어드민 계정 로그인 |
| 관리자 페이지 | `/admin` | 관리자 | 게시물, 댓글, 대댓글 관리 |

## 포함 기능

- 회원가입, 로그인, 로그아웃
- 프로필 조회 및 수정
- 게시글 생성, 조회, 수정, 삭제
- 게시글 조회수
- 댓글 및 대댓글
- 댓글 및 대댓글 좋아요/싫어요
- 관리자 전용 게시물 관리
- 관리자 전용 댓글/대댓글 관리
- 모바일 반응형 UI
- 공룡 캐릭터 기반 인터랙션
- Codex 내장 이미지 생성 기능을 활용한 프로필 이미지, 배경 이미지, 로고 제작

## 추후 기능

- 소셜 로그인
- 이메일 인증
- 비밀번호 찾기
- 이미지 업로드
- 알림 기능
- 관리자 통계 대시보드
- CI/CD 및 배포 자동화
