# 페이지 구성 체크리스트

## 생성 완료

- `/` 홈
- `/login` 로그인
- `/signup` 회원가입
- `/reset-password` 비밀번호 재설정
- `/auth/kakao` 카카오 로그인 진입
- `/saju` 정통 사주 입력
- `/tarot` 오늘의 타로
- `/search` 운세 검색
- `/analysis/loading` AI 분석 로딩
- `/result` 분석 결과
- `/history` 분석 기록
- `/my` 마이페이지
- `/profile/edit` 프로필 수정

## 이후 연결할 기능

- Supabase Auth 이메일 로그인/회원가입
- Supabase Kakao OAuth
- Supabase profile table
- Supabase Storage bucket 이미지 업로드
- OpenAI API 분석 요청
- 분석 결과 저장/조회
